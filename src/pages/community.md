## Contributing

We welcome contributions of all kinds:

| Type | Examples |
|------|----------|
| **Code** | Bug fixes, new features, performance improvements |
| **Documentation** | Guides, tutorials, API references |
| **Configurations** | Share your production-ready YAML configs |
| **Testing** | Bug reports, edge case identification |

### Getting Started

```bash
# Clone the repository
git clone https://github.com/ShedBoxAI/ShedBoxAI.git

# Install dependencies
pip install -e ".[dev]"

# Run tests
pytest
```
