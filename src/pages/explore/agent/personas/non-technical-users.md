# ShedBox Agent for Non-Technical Users

Access and analyze data using plain English—no coding required.

## Data Access for Everyone

You shouldn't need to know SQL, Python, or complex tools to answer questions about your business data.

**ShedBox Agent understands natural language and translates it into data operations.**

## How It Works

### Just Ask Questions

```
You: "How many customers signed up last month?"

ShedBox Agent:
✓ Connects to your database
✓ Runs the appropriate query
✓ Returns: "1,247 customers signed up in November 2024"
```

### Follow Up Naturally

```
You: "How does that compare to the previous month?"

ShedBox Agent:
✓ Remembers context
✓ Runs comparison
✓ Returns: "That's a 15% increase from October (1,084 signups)"
```

### Get Visualizations

```
You: "Show me this as a chart"

ShedBox Agent:
✓ Creates line chart of monthly signups
✓ Highlights the growth trend
```

## No Technical Skills Needed

## Example Conversation

```
You: "I need to understand our customer churn"

ShedBox Agent: "I can help with that. I'll connect to your customer database.
What time period would you like to analyze?"

You: "The last 6 months"

ShedBox Agent:
✓ Analyzing customer data for July-December 2024

Here's what I found:
- Total churned customers: 234
- Churn rate: 4.2% (down from 5.1% in the previous period)
- Most common churn reasons:
  1. Price (38%)
  2. Switched to competitor (28%)
  3. No longer needed (22%)
  4. Support issues (12%)

Would you like me to:
1. Break this down by customer segment?
2. Show the trend over time?
3. Compare to industry benchmarks?

You: "Show me the trend over time"

ShedBox Agent:
✓ Creating monthly churn visualization
[Shows chart with declining churn trend]

Good news: Your churn rate has decreased each month since August.
```
